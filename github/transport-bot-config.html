📊 Duhe Scoring Preview
duheScore = (priceScore + speedScore + capacityScore) × (reliability / 100)

priceScore = ((budget − bidPrice) / budget) × 100
speedScore = (requiredSpeed / bidSpeed) × 100
capacityScore = (bidCapacity / requiredCapacity) × 100
Lower bid price = higher priceScore. Faster delivery = higher speedScore. More capacity = higher capacityScore. Reliability multiplies everything.
